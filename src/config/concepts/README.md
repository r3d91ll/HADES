# Concept Configurations

## Overview
Configuration files for concept processing modules including ANT validation, LCC classification, query-as-node, supra-weight calculation, and temporal relationships.

## Structure
- `ant_validator.yaml` - Actor-Network Theory validation settings
- `lcc_classifier.yaml` - LCC (Labeling of Connected Components) configuration
- `query_as_node.yaml` - Query-as-node conversion settings
- `supra_weight_calculator.yaml` - Multi-dimensional weight calculation
- `temporal_relationships.yaml` - Time-based relationship configurations
- `theory_practice_bridge.yaml` - Theory to practice mapping settings

## Common Settings
- Validation thresholds
- Classification parameters
- Weight calculation methods
- Temporal window sizes
