# API Configurations

## Overview
Configuration files for the FastAPI server and PathRAG engine API endpoints.

## Structure
- `server.yaml` - FastAPI server settings (host, port, workers, etc.)
- `pathrag_engine.yaml` - PathRAG engine API configuration
- `base.yaml` - Base API settings shared across endpoints

## Common Settings
- Server host and port
- CORS configuration
- Authentication settings
- Rate limiting
- Request/response validation
- Logging levels
- Timeout values
