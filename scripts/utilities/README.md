## Usage Notes

Most of these scripts were created for specific one-time tasks and may no longer be relevant for current development. Before using any of these scripts, please review their content to ensure they still serve a useful purpose.

If a script is determined to be obsolete, consider moving it to the `dead-code` directory instead of deleting it, in accordance with the project's code organization standards.

## Maintenance

When creating new utility scripts:
1. Place them in this directory rather than the project root
2. Add appropriate documentation
3. Consider writing tests for any script that will be used regularly
